# Privacy Policy

**Last Updated:** April 7, 2026

Thank you for using **Keuanganku**. Your privacy and data security are our top priorities. This document explains how we manage, use, and protect your information when using the Keuanganku application.

**[Versi Bahasa Indonesia tersedia di bawah / Indonesian version is available below]**

## 1. Local Data Storage (Offline-First)

**Keuanganku is an offline-first application.** 

- **Financial Data**: All financial information you input into the app (such as transaction lists, amounts, wallets, budgets, and custom categories) is **stored entirely locally** on your device's internal storage using a SQLite database.
- **No Cloud Servers**: We **do not** transmit, copy, track, or store user financial data on external servers, cloud services, or any other third parties. You have 100% control over your data.

## 2. Analytics and Crash Data Collection

Currently, the application does not automatically collect or transmit usage analytics or crash reports to developer servers. If we introduce bug analysis features in the future to improve application quality, it will be strictly optional and we will request your explicit consent.

## 3. Device Permissions

To maximize its functionality, Keuanganku may request certain device permissions, such as:
- **Storage**: Used if you wish to use the database backup or restore options to/from the file system. You have the authority to decide where these files are placed.

## 4. Information Security

As we do not hold your financial data, the security of the data in this app relies on the operational security of your own device. To maintain your privacy:
- We highly recommend using screen lock features (such as passcode, PIN, or biometric recognition) on your device.
- Always keep your device's operating system updated to avoid vulnerabilities.

## 5. Changes to the Privacy Policy

We may revise this Privacy Policy from time to time in relation to potential new feature releases, such as optional cloud sync support. If such releases occur, we will post the updated policy and notify you via release notes or repository updates. Continued use after these changes signifies your acceptance of the new policy.

## 6. Contact Us

If you have any questions, concerns, feedback, or simply want to report an issue regarding the privacy handling principles in "Keuanganku", please create an issue/ticket on our official Repository page:
- **GitHub**: [dimassfeb-09/keuanganku](https://github.com/dimassfeb-09/keuanganku)

---
*By using and installing the Keuanganku application, you understand and agree to the privacy policy and data management as written in this document.*

# Kebijakan Privasi

**Terakhir Diperbarui:** 7 April 2026

Terima kasih telah menggunakan **Keuanganku**. Privasi dan keamanan data Anda adalah prioritas utama kami. Dokumen ini menjelaskan bagaimana kami mengelola, menggunakan, dan melindungi informasi Anda saat menggunakan aplikasi Keuanganku.

## 1. Penyimpanan Data Lokal (Offline First)

**Keuanganku adalah aplikasi berbasis lokal (offline-first).** 

- **Data Finansial**: Semua informasi keuangan yang Anda masukkan ke dalam aplikasi (seperti daftar transaksi, nominal, dompet/wallet, anggaran, dan kategori khusus) **disimpan sepenuhnya secara lokal** pada penyimpanan internal perangkat Anda menggunakan database SQLite.
- **Tidak Ada Server Cloud**: Kami **tidak** mengirimkan, tidak menyalin, tidak melacak, dan tidak menyimpan data finansial pengguna pada server eksternal, layanan komputasi awan (cloud), maupun pihak ketiga lainnya. Kendali atas data 100% berada di tangan Anda.

## 2. Pengumpulan Data Analitik atau Crash

Saat ini, aplikasi tidak mengumpulkan maupun mengirimkan data analitik penggunaan (usage analytics) ataupun data laporan kerusakan (crash reporting) secara otomatis ke server developer. Jika di masa depan kami menyediakan fitur analisis kesalahan untuk meningkatkan kualitas aplikasi, hal tersebut bersifat opsional dan kami akan meminta persetujuan eksplisit Anda.

## 3. Hak Akses (Permissions) Perangkat

Untuk memaksimalkan fungsinya, Keuanganku mungkin butuh meminta hak akses di perangkat Anda seperti:
- **Penyimpanan (Storage)**: Digunakan jika Anda ingin menggunakan opsi pencadangan (backup) atau pemulihan pangkalan data (restore database) ke dalam bentuk file sistem. Anda berwenang mengatur di mana berkas tersebut akan ditempatkan.

## 4. Keamanan Informasi

Karena kami tidak memegang data finansial Anda, keamanan data pada aplikasi ini bergantung pada tingkat keamanan operasional perangkat Anda sendiri. Demi menjaga privasi:
- Kami sangat merekomendasikan penggunaan fitur pengunci layar (screen lock) seperti passcode, PIN, atau pengenalan wajah/sidik jari pada perangkat Anda.
- Selalu perbarui sistem operasi perangkat Anda demi menghindari kerentanan.

## 5. Perubahan Kebijakan Privasi

Kami dapat merevisi Kebijakan Privasi ini dari waktu ke waktu terkait kemungkinan rilis fitur anyar, seperti dukungan *cloud sync* (pencadangan awan opsional) maupun fitur sejenis. Apabila rilis tersebut terwujud, kami akan menayangkan perbaharuan kebijakan dan memberitahukan Anda melalui catatan rilis aplikasi atau pembaruan di _repository_. Penggunaan yang dilanjutkan setelah perubahan tersebut menyatakan kesepakatan Anda atas kebijakan yang baru.

## 6. Hubungi Kami

Apabila Anda memiliki pertanyaan, kebingungan, umpan balik, atau sekadar ingin melaporkan masalah terkait dengan prinsip penanganan privasi pada "Keuanganku", silakan membuat isu/tiket (*issue*) melalui laman Repositori resmi kami:
- **GitHub**: [dimassfeb-09/keuanganku](https://github.com/dimassfeb-09/keuanganku)
